
 # Prerequisites
  ```$xslt
1. Docker installed on computer
```
# How to start
   
```$xslt
1. Enter root of project in terminal
```

```$xslt
2. docker login
```

```$xslt
3. docker build -t msi .
```

```$xslt
4. docker-compose up
```

This will start service on port 8080
