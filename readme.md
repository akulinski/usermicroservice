
 # Prerequisites
  ```$xslt
1. Docker installed on computer
2. jdk 8
3. maven
```
# How to start
   
```$xslt
1. Enter root of project in terminal
```
```$xslt
2. mvn clean install
```
```$xslt
3. docker login
```

```$xslt
4. docker build -t msi .
```

```$xslt
5. docker-compose up
```

This will start service on port 8080
